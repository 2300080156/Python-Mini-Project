# Python Mini Project

This repository contains solutions for the Python Mini Project assignment. The project demonstrates the use of Python fundamentals including variables, functions, conditional statements, loops, collections, exception handling, and object mutability concepts.

## Project Structure

```
Python_Mini_Project/
│
├── q1_result_analyzer.py
├── q2_library_system.py
├── q3_shopping_cart.py
├── discussion_points.py
└── README.md
```

## Question 1 – Student Result Analyzer

Features:
- Stores student information
- Calculates total and average marks
- Assigns grades using conditional statements
- Identifies subjects with marks below 40
- Uses functions and loops

## Question 2 – Library Book Management System

Features:
- Add books to a catalog
- Register library members
- Borrow and return books
- Display available books
- Uses dictionaries, tuples, lists, and sets

## Question 3 – Shopping Cart & Mutable Default Arguments

Features:
- Demonstrates the mutable default argument issue
- Provides the corrected implementation
- Implements a shopping cart system
- Calculates discounted totals
- Demonstrates tuple immutability and exception handling

## Discussion Topics

- Mutable vs Immutable objects
- Mutable default arguments
- Rebinding vs Mutating
- Function argument behavior in Python

## Technologies Used

- Python 3.x
- Functions
- Lists
- Tuples
- Dictionaries
- Sets
- Loops
- Conditional Statements
- Exception Handling

## How to Run

Clone the repository:

```bash
git clone <repository-url>
```

```bash
cd Python_Mini_Project
```

Run any file:

```bash
python q1_result_analyzer.py
python q2_library_system.py
python q3_shopping_cart.py
```
